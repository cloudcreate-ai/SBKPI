# SBKPI 游戏设定

本仓库中的 **SBKPI** 是一套与 **SBTI人格测试** 配套的**扑克牌面图鉴**：每张牌对应一种职场讽刺向人格原型，正面为点数与花色，背面为「黑话」与一条**技能描述**（偏桌游/梗向文案，非严谨规则书）。

## 牌组构成

- **54 张**：标准扑克 52 张 +大王、小王（Joker）。
- **数据来源**：`[persona-poker-deck.csv](./persona-poker-deck.csv)`（`card_id`、点数 `rank`、花色 `suit`、颜色、身份、人格代码与名称、风味文本、技能一行）。

## 颜色与身份


| 颜色     | 含义                   |
| ------ | -------------------- |
| **红色** | **Boss**（领导侧原型）      |
| **黑色** | **同事**（Coworker 侧原型） |


同一人格类型可能在红/黑两侧各出现一张（或按牌面矩阵分布），便于对照「同一原型在不同站位」的梗。

## 花色与矩阵（桌面端展示）

在图鉴页（`[sbkpi.html](../sbkpi.html)`）中，普通牌按矩阵排列，便于扫视：

1. **同一行**：相同点数（A、2…K）。
2. **同一列**：相同花色；列顺序为 ♥、♠、♦、♣（Hearts、Spades、Diamonds、Clubs）。
3. 相邻位置关系上，部分设计意图为**同人格**成对出现（具体以 CSV 为准）。

Joker 固定排在列表末尾（大王在前、小王在后）。

## 与 SBTI 人格图鉴的关系

- **SBTI 人格图鉴**（`[wiki.html](../wiki.html)`）：类型的文字介绍、海报与详情弹层。
- **SBKPI 图鉴**：同一批人格在「扑克技能条」语境下的**短描述**，与 wiki 通过页面内链接互跳；亦支持地址栏 `#人格代码` 或 `#poker-牌面ID` 定位（见各页说明）。

人格代码（`persona_code`）与语言包、图鉴一致，便于检索与对照。

## 技能与玩法说明

- CSV 中 **skill** 列为**风味技能**：多用「收益 / 费用 / 回合 / 手牌」等桌游语感，**未在本站实现为可执行规则**。
- 若你要线下自 house rule，可自行约定：例如将技能当作**一次性被动**、**每回合触发一次**或**纯 flavor不生效**，以桌边共识为准。
- 曾讨论过的**完整职场卡牌玩法**（KPI、R-D-C-T、四槽一增益等）见设计备忘：[sbkpi-play-design-discussion.md](./sbkpi-play-design-discussion.md)（**非**本站已实现规则）。
